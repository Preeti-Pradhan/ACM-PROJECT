The Hinglish folder has 4 files in conll format - train data, validation data, test data (unlabeled), test labels
The spanglish folder has 2 files in conll format - train data, validation data, test data unlabeled


For details on data collection and annotation please refer to the task description paper: 'SemEval-2020 Task 9: Overview of Sentiment Analysis of Code-Mixed Tweets'
for test labels of spanglish data please refer to the paper : 'LinCE: A Centralized Benchmark for Linguistic Code-switching Evaluation' (https://arxiv.org/abs/2005.04322)
